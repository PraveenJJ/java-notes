# Java Training Sessions - Notes & Examples

Welcome to the **Java Training Sessions** repository! This repository contains a structured collection of notes, examples, and diagrams from Java learning sessions. 

## 📂 Repository Structure

The repository is organized into two main sections:

### `Class-Notes/`
This directory contains comprehensive theoretical notes covering various Java concepts. The notes are organized by topic for easy reference.

Current topics include:
* **1. Java Platform Fundamentals**
* **2. Variables and Data Types**

### `Class-Examples/`
This directory contains practical code examples, demonstrations, and visual diagrams discussed during the sessions. It is organized chronologically by date.

For instance:
* **09-03-2026**: Contains examples like `Animal.java` and illustrations such as `variable-scope.png`.

## 🚀 Getting Started

If you are a student or someone following along:
1. Start by reading the markdown files inside `Class-Notes/` to grasp the theoretical concepts.
2. Cross-reference the theory with practical implementation by checking the source files in `Class-Examples/`.
3. You can compile and run the Java examples using any standard IDE (like IntelliJ IDEA, Eclipse, VS Code) or the command line:
   ```bash
   javac Animal.java
   java Animal
   ```
