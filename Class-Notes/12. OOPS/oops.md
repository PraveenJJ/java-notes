# Object Oriented Programming (OOP)

## Definition
- OOPs is a programming paradigm that organizes software design around objects instead of functions. 
- Objects represent real-world entities that contain both data (fields) and behavior (methods). 
- Java uses OOPs principles to make programs modular, reusable, secure, and easy to maintain.

---

## Why OOP Exists (Problem It Solves)

Before OOP, large software systems had problems:
- Difficult to maintain
  - Code duplication
  - Poor modularity
  - Difficult collaboration in teams
- OOP solves this by providing:
  - `Encapsulation` → data protection
  - `Inheritance` → code reuse
  - `Polymorphism` → flexible behavior
  - `Abstraction` → hiding complexity
